# 🐍 Snake Game

A classic Snake game built with vanilla JavaScript, HTML5 Canvas, and CSS.

## Features

- 🎮 **Classic Gameplay**: Move the snake around and eat food
- 📊 **Score Tracking**: Keep track of your score and high score
- ⚡ **Progressive Difficulty**: Game speed increases as you eat more food
- 🎨 **Modern UI**: Beautiful gradient design with smooth animations
- 💾 **High Score Persistence**: Your best score is saved in browser storage
- ⏸️ **Pause Feature**: Pause and resume the game anytime

## How to Play

1. **Start the Game**: Click the "Start Game" button
2. **Move the Snake**: Use Arrow Keys (↑ ↓ ← →) to control the snake
3. **Eat Food**: Guide the snake to eat the red circles
4. **Grow Longer**: Each food eaten makes the snake longer
5. **Avoid Collision**: Don't hit the walls or yourself
6. **Pause/Resume**: Click "Pause" to pause the game

## Controls

| Key | Action |
|-----|--------|
| ↑ Arrow Up | Move Up |
| ↓ Arrow Down | Move Down |
| ← Arrow Left | Move Left |
| → Arrow Right | Move Right |
| Start Button | Begin Game |
| Pause Button | Pause/Resume |
| Reset Button | New Game |

## Scoring

- **+10 points** for each food eaten
- **Difficulty increases** every time you eat food
- **High score** is automatically saved

## Game Rules

- Snake wraps around screen edges
- Game ends if snake collides with itself
- Each food eaten increases game speed slightly
- Try to get the highest score possible!

## Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start playing!
